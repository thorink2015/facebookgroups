# shared/ — your drop folder for me

**Put any file you want me to use here.** CSVs, copy, notes, screenshots, new
research, image briefs — anything. When you say "go through what I shared," this is
the first place I look.

Tips that make it efficient:
- Give files clear names (e.g. `advertiser-prospects-with-hooks.csv`).
- A one-line note at the top of a doc ("this replaces the old templates") saves a lot
  of back-and-forth.
- Big binary stuff (lots of images) is better uploaded straight into the dashboard's
  **Images** tab — that stores + tags them for rotation.
