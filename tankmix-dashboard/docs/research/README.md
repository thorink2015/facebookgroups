# docs/research/ — upload your docs here

This is the folder to drop your own files into (via the GitHub web "Add file →
Upload files" button, or by copying them in locally and committing):

- your raw research documents (the stealth/auto-poster research, etc.),
- the **Tank Mix posting playbook** (`tank-mix-facebook-posting-playbook-v1.md`),
- your group CSVs if you want them version-controlled,
- anything else worth keeping with the project.

These files are kept in git (not ignored), so they'll be saved to GitHub.
If any contain things you'd rather not commit, just don't upload those.
